# Example Reference Skill

## Overview

Use this as a tiny example of the preferred skill folder shape.

2. Read this file for the core workflow.
3. Open the checklist in [references/checklist.md](references/checklist.md) when you want a concise reference.

## Output Contract

- the split between `SKILL.md` and `references/`
